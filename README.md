# Dynamic Data Table Manager

A Next.js, Redux Toolkit, and Material UI based web application for dynamic data management. It includes real-world features such as sorting, searching, pagination, CSV import/export, inline editing, and theme customization.

---

## Features

- Default columns: Name, Email, Age, Role
- Global search across all table fields
- Sorting with ascending and descending toggle
- Client-side pagination (10 rows per page)
- Manage Columns: add, show, or hide dynamically
- CSV Import and Export (supports visible columns only)
- Inline editing with input validation
- Row deletion with confirmation
- Light/Dark theme toggle using Material UI
- Row reordering via drag-and-drop
- Responsive layout optimized for all screen sizes

---

## Tech Stack

- Next.js 14 (App Router)
- Redux Toolkit
- Material UI v5
- React Hook Form
- PapaParse and FileSaver.js for CSV operations
- @dnd-kit/core for drag-and-drop functionality
- TypeScript
